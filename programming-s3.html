Bouncy Balls

The “BouncyBalls” project is a demonstration of how objects can be used in JavaScript. The program is to have multiple balls which would bounce off the edges of the screen. The balls will change color whenever they collide however they will not interrupt their motion.

“BouncyBalls” utilizes Canvas API, used to draw graphics via Javascript and the HTML <canvas> element. Here it is used to display an animation. The ball properties are stored in an object called Ball like the size and color. This is made to be referenced by Ball.prototype.draw. This function will draw the ball with the given properties and their values. Ball.prototype.update makes sure that the ball will not pass further than the canvas. The while (balls.length < 25) function will generate balls with random sizes and colors until there are 25 of them. A ball created from that function will be pushed into the balls array. Lastly, requestAnimationFrame(loop) will repeatedly run the loop.

The challenge that came with it was just trying to piece together the code, and to visualize what I wanted to achieve. It’s easy to get lost if I don’t have a clear vision in my head of my goal. Doing so will make it easier to break the process down into small steps that are easy to accomplish.

I learned how to investigate, looking up certain functions or pieces of code I didn’t know helped me a lot. I had read through the document carefully in order to understand what I was supposed to do.
